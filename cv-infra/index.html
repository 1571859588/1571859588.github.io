    Projects

    RAG-EDA: Scalable Document QA Infrastructure for EDA Tools 2024.10 - 2025.07
    Core Developer
    A production-grade, containerized RAG system with hybrid retrieval, deployed at Empyrean Technology
    • Architected full-stack RAG pipeline (parsing → FAISS+BM25 hybrid retrieval → FlagLLM Reranker → Qwen1.5-14B) for 5 tool knowledge bases
    • Designed multi-channel retrieval with deduplication and metadata filtering; fine-tuned Reranker for domain-specific re-scoring, boosting Top-5 hit rate from 60% to 90%+
    • Built monitoring and automated evaluation framework covering retrieval recall, reranker accuracy, and end-to-end QA metrics with auto-generated reports
    • Containerized entire stack (Streamlit frontend, MySQL backend, retrieval engine) with Docker; implemented user auth, multi-KB switching, and session management
    High-Throughput PDF Parsing Pipeline for EDA Documents 2026.01 - 2026.05
    Core Developer
    A configurable, high-throughput PDF parsing pipeline with multi-tool orchestration and auto-verification
    • Designed unified JSON Schema IR layer supporting 6 element types; orchestrated MinerU and pdf2kg parsers with bbox merging for robust extraction
    • Built OpenCV table recognition with cell merging and cross-page stitching (Markdown/HTML output); designed caption-figure association via font style analysis
    • Implemented LLM-enhanced chunking with ancestor summaries and token budget control; dual outline recognition via PDF bookmarks + font-size heuristics
    • Deployed configurable Flask pipeline with Docker; built auto-verification loop to detect and re-parse missing elements
