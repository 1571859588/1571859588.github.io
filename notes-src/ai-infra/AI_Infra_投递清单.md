## 投递策略

### 优先级排序（建议投递顺序）
1. **无问芯穹 / 硅基流动**（最对口，技术深度高，转正机会大）
2. **智谱 / 月之暗面**（AI独角兽，技术好）
3. **字节 / 阿里**（大厂背书+高薪+转正）
4. **NVIDIA**（天花板，但竞争激烈）
5. **华为昇腾**（国产替代主线，稳定）
6. **其他**（兜底）

### 投递节奏
- **2027年3-5月**：投暑期实习，主攻前3梯队，争取转正offer
- **2027年8月**：秋招提前批，全梯队投递
- **2027年9-11月**：正式批，补投+谈薪

### 简历关键词（HR筛选用）
务必在简历中体现：
- CUDA / GPU编程
- vLLM / DeepSpeed / Megatron（至少一个源码级理解）
- PagedAttention / FlashAttention / ZeRO（能讲原理）
- C++ / Python / 分布式系统
- 性能优化 / benchmark / 算子开发

### 面试准备重点
- 源码级问答：vLLM调度器、PagedAttention实现
- 手撕CUDA：矩阵乘法、reduce、softmax kernel
- 系统设计：设计高吞吐LLM推理服务
- 通信原语：all-reduce通信量计算、TP/PP权衡

- **实习期间**：如果暑期实习拿到大厂offer，建议从港中文公司退出
