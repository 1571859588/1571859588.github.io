## 代码环境

- **硬件**：任意 NVIDIA GPU（V100 / A100 / H100 / RTX 30xx / RTX 40xx 均可）
- **软件**：CUDA Toolkit 11.x+、PyTorch 2.x（含 `torch.compile`）、Triton
- **sudo 权限**：本系列假设用户无 sudo 权限，所有命令均为用户态可执行
  - nsys / ncu 基础功能✅无需 sudo
  - 部分硬件性能计数器⚠️受 `NVreg_RestrictProfilingToAdminUsers` 限制，详见 `5. 工具链与扩展篇/2. 性能分析工具链.md`
- **多版本管理**：用户态装多个 CUDA Toolkit 共存，详见 `2. 基础篇/1. CUDA开发环境搭建.md`
- 所有代码可直接复制运行，附预期输出
