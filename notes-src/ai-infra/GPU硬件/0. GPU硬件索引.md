## 代码环境

- **硬件**：任意 NVIDIA GPU（V100 / A100 / H100 / RTX 30xx / RTX 40xx 均可）
- **软件**：nvidia-smi（驱动自带，无需额外安装）、PyTorch 2.x、（可选）TensorRT/trtexec
- **sudo 权限**：本系列假设用户无 sudo 权限（WSL + 有 GPU 的 Linux 服务器）
  - 除 `nvidia-smi mig -cgi`（创建 MIG 实例）⚠️ 需 sudo 外，其余命令均 ✅ 无需 sudo
  - MIG 创建的无 sudo 替代说明详见 `3. 架构演进篇/2. NVIDIA-GPU架构演进史.md`
  - WSL 下 `nvidia-smi topo -m` / `nvlink -s` 可能输出受限，建议在真实 Linux 服务器上运行
- **命令权限标注规范**：每条命令前用 `✅ 无需 sudo` 或 `⚠️ 需 sudo` 注释
