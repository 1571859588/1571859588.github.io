## 代码环境

- **硬件**：任意 NVIDIA GPU（单卡可跑通大部分概念验证；多卡效果更佳）
- **用户环境**：Ubuntu Linux 服务器，**8 张 A100 40G**，**无 sudo 权限**
- **软件**：PyTorch 2.x + NCCL（PyTorch 自带）+ Python 3.10+（通过 conda 管理）
- **sudo 权限说明**（⚠️ 重要，先读）：
  - 所有命令已标注权限：`✅ 无需 sudo` / `⚠️ 需 sudo`
  - `nvidia-smi`：✅ 无需 sudo（查看 GPU 状态、拓扑）
  - `torchrun`：✅ 无需 sudo（启动分布式训练）
  - NCCL 环境变量：✅ 无需 sudo（`export NCCL_DEBUG=INFO` 等）
  - PyTorch/NCCL 安装：✅ 用 `conda install` 或 `pip install`，无需 sudo
  - IB 网卡查看（`ibstat`）：⚠️ 需 root 或 `ibutils` 包安装（通常系统已装则 ✅）
  - GPUDirect RDMA：⚠️ 需 root 加载内核模块，无 sudo 不可用
  - `/dev/shm` 扩容：⚠️ 需 root，Docker 容器内需 `--shm-size`，无 sudo 则用 `NCCL_SHM_DISABLE=1` 绕过
- **多机前提**：跨机实验需要多台机器 + SSH 免密登录 + 同一 conda 环境
