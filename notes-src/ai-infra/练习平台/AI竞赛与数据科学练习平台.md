## 二、Kaggle（全球首选，必注册）

**网址**：https://www.kaggle.com/

### 2.1 为什么是首选

- **全球最大数据科学社区**，Google 旗下
- **免费 GPU**：双 T4（30GB 显存），每周 30 小时
- **真实数据集**：海量公开数据集可直接用
- **竞赛有奖金**：顶级竞赛奖金 $10,000-$100,000
- **Notebook 社区**：大量公开代码可学习
- **国内可直连**，无需翻墙

### 2.2 核心功能

| 功能 | 说明 |
|------|------|
| **Competitions** | 数据科学竞赛，有奖金或纯练习 |
| **Datasets** | 公开数据集，可直接加载到 Notebook |
| **Notebooks** | 在线 Jupyter，免费 GPU/TPU |
| **Models** | 预训练模型库（类似 HuggingFace） |
| **Courses** | 免费入门课程（Python/ML/DL/SQL） |
| **Discussion** | 社区讨论，竞赛解题思路 |

### 2.3 推荐入门路径

```
1. 注册账号（邮箱即可）
2. 做 Courses 里的免费入门课
   - Python → 机器学习 → 深度学习 → Pandas
3. 做 Getting Started 竞赛（无门槛，纯练习）
   - Titanic（二分类入门）
   - House Prices（回归入门）
   - Digit Recognizer（CV 入门）
4. 看 Discussion 和公开 Notebook 学大佬思路
5. 参加 Featured 竞赛（有奖金，竞争激烈）
```

### 2.4 免费 GPU 使用

```python
# 开启 GPU：Settings → Accelerator → GPU T4 x2
# 开启 TPU：Settings → Accelerator → TPU v3-8

# 查看分配的 GPU
!nvidia-smi

# 查看 GPU 数量
import torch
print(torch.cuda.device_count())  # 2

# 安装额外依赖
!pip install vllm transformers
```

**限制**：
- 每周 30 小时 GPU 额度
- 每次最长 12 小时，超时自动断开
- Notebook 不活跃 60 分钟会断开
- 数据集存储 100GB

### 2.5 适合的 AI-Infra 阶段

| 阶段 | 用法 |
|------|------|
| 三（CUDA 验证） | 免费 T4 验证 `torch.cuda.is_available()` |
| 三（小模型训练） | 跑 7B 模型微调实验 |
| 五（推理实验） | 用 Ollama/vLLM 跑小模型 |
| 学习 RAG | 用 Kaggle 数据集做文档问答 |

## 三、天池（国内首选）

**网址**：https://tianchi.aliyun.com/

### 3.1 核心优势

- **阿里云旗下**，国内最大数据科学竞赛平台
- **国内访问快**，无需翻墙
- **竞赛有奖金**：顶级赛事奖金可达百万
- **免费算力**：阿里云 PAI/DSW 提供有限免费 GPU
- **中文社区**：讨论区、教程全中文

### 3.2 核心板块

| 板块 | 说明 |
|------|------|
| **竞赛** | 算法竞赛（CV/NLP/推荐/时序） |
| **实验室** | 免费 GPU Notebooks（阿里云 DSW） |
| **学习赛** | 入门练习赛，无门槛 |
| **技术圈** | 中文技术社区 |
| **课程** | 免费系统课程 |

### 3.3 推荐入门路径

```
1. 注册（阿里云账号/手机号）
2. 做"学习赛"入门
   - 零基础入门系列（二分类/回归）
   - 图像分类入门
   - NLP 入门
3. 用实验室的免费 GPU 跑 Notebook
4. 参加正式竞赛
```

### 3.4 免费 GPU

- 阿里云 DSW（Data Science Workshop）
- 免费额度有限（通常每天几小时）
- GPU 型号不固定（T4/V100）
- 适合轻量实验

## 四、和鲸社区

**网址**：https://www.heywhale.com/

### 4.1 特点

- 国内数据科学社区
- 免费 Notebook 环境（ModelWhale）
- 有练习赛和数据集
- 中文友好

### 4.2 适合

- 国内用户想找中文 AI 社区
- 想做练习赛但不想到 Kaggle 翻英文
- 需要免费 GPU 跑小实验

## 五、DrivenData（AI 解决社会问题）

**网址**：https://www.drivendata.org/

### 5.1 特点

- 竞赛主题是用 AI 解决社会问题（医疗/环境/农业）
- 题目有社会意义，适合写简历
- 国际平台，需翻墙

### 5.2 适合

- 想做有社会影响力的项目
- 简历上写"用 AI 解决 XX 社会问题"比纯刷题更有亮点

## 六、AIcrowd（国际 AI 竞赛）

**网址**：https://www.aicrowd.com/

### 6.1 特点

- 国际 AI 竞赛平台，和 Kaggle 类似
- 竞赛偏研究向（RL/Multi-Agent 等）
- 有公开数据集和基准测试

### 6.2 适合

- 想参加偏研究的竞赛
- 对强化学习/多智能体感兴趣

## 七、平台选型建议

```
你的目标是什么？
│
├─ 纯学习入门
│   ├─ 能看英文 → Kaggle Courses（免费）
│   └─ 只看中文 → 天池课程 + 和鲸社区
│
├─ 打竞赛赚奖金/写简历
│   ├─ 国际 → Kaggle Featured 竞赛
│   └─ 国内 → 天池正式竞赛
│
├─ 找真实数据集做项目
│   └─ Kaggle Datasets（最全）
│
├─ 练 RAG / 大模型应用
│   ├─ Kaggle（免费 GPU + 数据集）
│   └─ 天池实验室
│
└─ 用 AI 解决社会问题
    └─ DrivenData
```

## 八、竞赛对求职的价值

| 竞赛级别 | 简历价值 | 说明 |
|----------|----------|------|
| Kaggle 银牌+ | ⭐⭐⭐⭐ | 大厂 AI 岗位加分项 |
| Kaggle 铜牌 | ⭐⭐⭐ | 证明有实战经验 |
| 天池 Top 10% | ⭐⭐⭐ | 国内 AI 公司认可 |
| 天池参与奖 | ⭐⭐ | 有总比没有好 |
| 学习赛完成 | ⭐ | 简历可提，但非核心 |

> **注意**：竞赛是锦上添花，不是必需项。AI-Infra 岗位更看重项目经验（阶段六的两个项目），竞赛是加分项不是门槛。
