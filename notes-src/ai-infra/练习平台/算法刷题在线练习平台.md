## 二、LeetCode 力扣（首选，必刷）

**网址**：https://leetcode.cn/

### 2.1 为什么是首选

- **题库最全**：3000+ 题，覆盖所有常见算法题型
- **大厂原题多**：字节/腾讯/阿里面试算法题基本都能找到
- **题解社区活跃**：每题都有中英文题解，含多种解法
- **企业题库**：有按公司整理的高频题（LeetCode Premium 功能，但社区有免费整理版）

### 2.2 推荐刷题顺序

| 顺序 | 题单 | 题数 | 目标 |
|------|------|------|------|
| 1 | **LeetCode Hot 100** | 100 题 | 高频必刷，覆盖核心题型 |
| 2 | **面试经典 150 题** | 150 题 | 官方精选，Hot 100 的补充 |
| 3 | **剑指 Offer** | 75 题 | 经典面试题（牛客上也有） |
| 4 | **字节跳动专题** | ~200 题 | 投字节必刷 |
| 5 | **每日一题** | 持续 | 保持手感 |

### 2.3 高频题型分布（Hot 100 统计）

| 题型 | 占比 | 代表题 |
|------|------|--------|
| 数组/哈希表 | 25% | 两数之和、子数组和为 K |
| 双指针/滑动窗口 | 15% | 三数之和、最长无重复子串 |
| 链表 | 15% | 反转链表、合并有序链表 |
| 二叉树/DFS | 20% | LCA、层序遍历 |
| 动态规划 | 15% | 最长递增子序列、编辑距离 |
| 其他（栈/堆/图） | 10% | 有效括号、拓扑排序 |

### 2.4 使用技巧

- **先按题型刷**，不要随机刷（同题型集中突破效率高）
- **先简单后中等**，Hard 视目标公司决定是否刷
- **每题先想 15 分钟**，想不出看题解，理解后自己重写一遍
- **用 Python 刷**（AI 方向面试通常用 Python，代码短效率高）

### 2.5 LeetCode 旗下赛道板块 + LeetXXX 衍生平台

LeetCode 主站内还有多个专项赛道（非独立域名，共享主站账号）：

| 赛道 | 定位 |
|------|------|
| LeetDatabase | SQL 专项 |
| LeetShell | Shell 脚本 |
| LeetConcurrency | 多线程并发编程 |
| LeetPandas | Pandas 数据处理 |
| LeetDesign | 系统/OOD 设计 |

此外还有一批**独立第三方 LeetXXX 刷题站**（LeetGPU、LeetCUDA、LeetSQL 等），尤其 **LeetGPU/LeetCUDA 是 AI Infra 推理优化方向的核心刷题平台**。

> 📌 详细整理见本目录 [`LeetXXX系列刷题平台整理.md`](./LeetXXX系列刷题平台整理.md)

## 三、牛客网（国内面试必备）

**网址**：https://www.nowcoder.com/

### 3.1 核心优势

- **剑指 Offer 原题**：75 道经典面试题，国内面试覆盖率极高
- **公司真题卷**：字节/腾讯/阿里/美团等历年笔试真题
- **笔试模拟**：限时模拟真实笔试环境
- **讨论区**：面试面经分享

### 3.2 推荐题单

| 题单 | 说明 |
|------|------|
| **剑指 Offer** | 75 题，必刷，面试原题率极高 |
| **字节跳动 2024 校招** | 字节笔试真题 |
| **腾讯 2024 校招** | 腾讯笔试真题 |
| **算法笔试全真模拟** | 限时模拟，练手感 |

### 3.3 配合 LeetCode 使用

- LeetCode 刷题型基础 → 牛客刷公司真题卷
- 牛客的剑指 Offer 和 LeetCode 的剑指 Offer 题目相同，选一个平台刷即可

## 四、AcWing（系统学算法）

**网址**：https://www.acwing.com/

### 4.1 适合谁

- 算法基础薄弱，想系统学习
- 目标是蓝桥杯/ACM 竞赛
- 喜欢有课程引导的学习方式

### 4.2 核心内容

| 课程 | 内容 | 价格 |
|------|------|------|
| **算法基础课** | 排序、二分、DP、图论等基础 | 付费（有免费试听） |
| **算法提高课** | 进阶算法 | 付费 |
| **蓝桥杯辅导课** | 竞赛专项 | 付费 |

### 4.3 题库特点

- 题目按算法分类（不像 LeetCode 按难度）
- 每题有视频讲解
- 适合"学一个算法刷一类题"

## 五、其他平台简评

### 5.1 Codeforces（国际竞赛）

**网址**：https://codeforces.com/

- 国际顶级算法竞赛平台
- 题目难度高，适合 ACM 选手
- 有定期比赛（Round），可提升排名
- **不适合面试准备**，除非目标是对冲基金/量化

### 5.2 HackerRank（外企面试）

**网址**：https://www.hackerrank.com/

- 国际化平台，外企（Google/Meta/Amazon）面试常用
- 有企业认证考试
- 题目偏工程化（不只是纯算法）
- **投外企必刷**，投国内大厂用 LeetCode 即可

### 5.3 CodeWars（趣味入门）

**网址**：https://www.codewars.com/

- 游戏化设计，解题升级
- 题目小巧，适合碎片时间
- 多语言支持（Python/Go/JS 等）
- **适合零基础入门**，进阶后转 LeetCode

## 六、刷题策略建议

### 6.1 按目标公司

| 目标 | 平台 | 题单 | 题数 |
|------|------|------|------|
| 字节跳动 | LeetCode | 字节专题 + Hot 100 | ~300 |
| 腾讯/阿里 | LeetCode + 牛客 | Hot 100 + 公司真题 | ~200 |
| 外企 | LeetCode + HackerRank | Hot 100 + HackerRank 认证 | ~200 |
| 中小公司 | LeetCode | Hot 100 | 100 |

### 6.2 按时间规划

| 时间 | 目标 | 每日题量 |
|------|------|----------|
| 3 个月 | 大厂面试 | 3-5 题/天 |
| 1 个月 | 冲刺面试 | 5-8 题/天（重质量） |
| 1 周 | 临时突击 | 只刷 Hot 100 高频题 |

### 6.3 按语言选择

| 语言 | 适合场景 | 刷题体验 |
|------|----------|----------|
| **Python** | AI 方向面试（推荐） | 代码短，写得快 |
| Java | 后端面试 | 代码长但规范 |
| Go | 云原生面试 | 简洁，适合并发题 |
