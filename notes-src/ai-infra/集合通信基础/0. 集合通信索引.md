## 代码环境

- **硬件**：任意 NVIDIA GPU（单卡即可跑通大部分代码；多卡效果更佳）
- **软件**：PyTorch 2.x + NCCL（PyTorch 自带）+ Python 3.10+
- **sudo 权限说明**（⚠️ 重要，先读）：
  - 本系列假设用户环境为 **WSL + 有 GPU 的 Linux 服务器，但无 sudo 权限**
  - 所有命令已标注权限：`✅ 无需 sudo` / `⚠️ 需 sudo` / `⚠️ WSL 不可用`
  - `nvidia-smi topo -m`：✅ 无需 sudo，但 WSL 下 NVLink 拓扑可能显示为 SYS/NODE
  - IB 网卡查看（`ibstat`/`ibv_devinfo`）：⚠️ WSL 通常无 IB 硬件，只能在真实服务器看
  - GPUDirect RDMA：⚠️ 需 root 加载内核模块，无 sudo 不可用
  - NCCL 安装：✅ 用 `conda install -c nvidia nccl`，无需 sudo
  - nccl-tests 编译：✅ 用 conda 装 CUDA/NCCL/MPI，make 指向 conda 路径
  - 多机 mpirun：✅ 无需 sudo，但需配置 SSH 免密
- **多机前提**：跨机实验需要多台机器 + SSH 免密登录 + 同一 conda 环境
