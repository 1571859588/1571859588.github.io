    AI_Infra_投递清单

    AI Infra 应届岗投递清单（2028届）

    适用：2027年暑期实习（3-5月投递）+ 2027年秋招（8-11月） 薪资数据基于2026届秋招公开信息，2028届预计持平或小幅上涨

    第一梯队：大厂（薪资高+背书强+转正机会大）

    1. 华为 — 昇腾/CANN/2012实验室

    • 岗位：AI计算工程师、异构计算工程师、大模型系统工程师
    • 方向：CANN算子开发、MindSpore框架、昇腾NPU适配
    • 薪资：天才少年100万+，普通应届35-55万
    • 特点：硬核底层，CUDA/C++要求高，国产替代主线
    • 加分：懂CUDA + 愿意做国产芯片生态
    • 投递：华为招聘官网，关注”天才少年”计划

    2. 阿里 — 通义实验室/达摩院/阿里云

    • 岗位：大模型推理工程师、AI Infra工程师、高性能计算工程师
    • 方向：推理框架（阿里有自研框架）、训练加速、PAI平台
    • 薪资：35-60万，阿里星A 80万+
    • 特点：技术氛围好，通义团队是头部大模型团队
    • 加分：有vLLM/DeepSpeed源码经验
    • 投递：阿里招聘官网，搜索”AI Infra”“大模型系统”

    3. 字节跳动 — 豆包/AI Lab/火山引擎

    • 岗位：大模型推理优化、训练框架工程师、AI Infra工程师
    • 方向：豆包大模型训练推理、veRL框架、火山引擎MaaS
    • 薪资：40-70万，顶尖SP 80万+
    • 特点：技术激进，卷但钱多，大模型投入最大之一
    • 加分：能讲清楚PagedAttention/FlashAttention实现
    • 投递：字节招聘官网，搜索”大模型系统”“推理优化”

    4. 腾讯 — 混元/TEG/微信

    • 岗位：大模型训练推理工程师、AI Infra工程师
    • 方向：混元大模型、Angel机器学习框架
    • 薪资：35-55万，SP 60万+
    • 特点：相对不那么卷，技术扎实
    • 投递：腾讯招聘官网

    5. 百度 — 文心/ACG

    • 岗位：大模型系统工程师、推理引擎工程师
    • 方向：文心一言训练推理、PaddlePaddle
    • 薪资：30-50万
    • 特点：大模型老牌玩家，技术积累深

    第二梯队：AI独角兽（技术纯度高+成长快）

    6. 智谱AI（Zhipu）

    • 岗位：AI Infra工程师、推理优化工程师
    • 方向：GLM系列训练推理、推理框架优化
    • 薪资：35-60万
    • 特点：国内大模型第一梯队，技术导向
    • 投递：智谱招聘官网/BOSS直聘

    7. 月之暗面（Moonshot）

    • 岗位：Infra工程师、大模型系统工程
    • 方向：Kimi推理优化、长上下文技术
    • 薪资：40-70万
    • 特点：长上下文技术领先，技术挑战大
    • 投递：官网/猎头

    8. MiniMax

    • 岗位：AI Infra、训练推理工程师
    • 薪资：35-60万
    • 特点：多模态大模型，技术栈全

    9. 百川智能（Baichuan）

    • 岗位：大模型系统工程师
    • 薪资：30-55万

    10. 阶跃星辰（StepFun）

    • 岗位：Infra工程师
    • 薪资：35-60万
    • 特点：多模态方向强

    第三梯队：AI Infra专业公司（最对口，技术深度最高）

    11. 无问芯穹（Infini-AI）

    • 岗位：AI Infra工程师、算子开发工程师
    • 方向：大模型推理加速、异构计算、MaaS平台
    • 薪资：35-60万
    • 特点：纯Infra公司，最对口，技术氛围极好
    • 加分：有CUDA/算子开发经验
    • 投递：官网/BOSS直聘，强烈推荐

    12. 硅基流动（SiliconFlow）

    • 岗位：推理引擎工程师、训练框架工程师
    • 方向：开源推理框架、大模型加速
    • 薪资：35-60万
    • 特点：袁进辉团队，技术顶尖，开源文化
    • 投递：官网，强烈推荐

    13. 潞晨科技（ColossalAI）

    • 岗位：分布式训练工程师、推理优化工程师
    • 方向：ColossalAI框架、视频生成Infra
    • 薪资：30-55万
    • 特点：尤洋团队，分布式训练专精
    • 加分：给ColossalAI贡献过PR是加分项
    • 投递：官网/BOSS直聘

    14. 后摩智能

    • 岗位：AI Infra工程师、算子开发
    • 方向：存算一体芯片软件栈
    • 薪资：30-50万
    • 特点：芯片+软件栈，适合懂底层的人

    第四梯队：芯片厂（薪资天花板，门槛高）

    15. NVIDIA（英伟达）

    • 岗位：CUDA工程师、TensorRT工程师、解决方案工程师
    • 薪资：50-100万+（外企薪资天花板）
    • 特点：Infra技术源头，最强背书
    • 加分：CUDA能力强，英语好
    • 投递：NVIDIA官网，关注校招+实习

    16. AMD

    • 岗位：AI软件工程师、ROCm工程师
    • 薪资：40-70万
    • 特点：挑战NVIDIA生态，机会多

    17. 地平线

    • 岗位：BPU算子开发、AI Infra工程师
    • 薪资：35-55万
    • 特点：车载AI芯片龙头

    18. 寒武纪

    • 岗位：算子开发、推理框架工程师
    • 薪资：30-50万
    • 特点：国产AI芯片先行者

    19. 壁仞科技 / 燧原科技 / 摩尔线程

    • 岗位：AI Infra、算子开发、软件栈工程师
    • 薪资：30-55万
    • 特点：国产GPU，机会多但稳定性存疑

    投递策略

    优先级排序（建议投递顺序）

    1. 无问芯穹 / 硅基流动（最对口，技术深度高，转正机会大）
    2. 智谱 / 月之暗面（AI独角兽，技术好）
    3. 字节 / 阿里（大厂背书+高薪+转正）
    4. NVIDIA（天花板，但竞争激烈）
    5. 华为昇腾（国产替代主线，稳定）
    6. 其他（兜底）

    投递节奏

    • 2027年3-5月：投暑期实习，主攻前3梯队，争取转正offer
    • 2027年8月：秋招提前批，全梯队投递
    • 2027年9-11月：正式批，补投+谈薪

    简历关键词（HR筛选用）

    务必在简历中体现：

    • CUDA / GPU编程
    • vLLM / DeepSpeed / Megatron（至少一个源码级理解）
    • PagedAttention / FlashAttention / ZeRO（能讲原理）
    • C++ / Python / 分布式系统
    • 性能优化 / benchmark / 算子开发

    面试准备重点

    • 源码级问答：vLLM调度器、PagedAttention实现
    • 手撕CUDA：矩阵乘法、reduce、softmax kernel
    • 系统设计：设计高吞吐LLM推理服务
    • 通信原语：all-reduce通信量计算、TP/PP权衡

    关于港中文公司工作的处理

    • 实习期间：如果暑期实习拿到大厂offer，建议从港中文公司退出
    • 退出话术：感谢培养，因学业/秋招压力，无法继续投入
    • 保留关系：港中文老师的人脉有价值，好聚好散
    • 简历处理：港中文的Agent项目保留，但重新包装为”大模型应用系统工程”

    数据说明

    • 薪资为2026届秋招公开数据（OfferShow、脉脉、知乎），2028届预计持平
    • SP/SSP需有强项目/实习/PR背书，普通offer为下限
    • 芯片厂薪资含股票，实际到手有波动
    • 国产芯片公司（寒武纪、壁仞等）稳定性需评估，建议作为备选
