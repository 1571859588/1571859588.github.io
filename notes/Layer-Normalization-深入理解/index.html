    Layer Normalization 深入理解

    1. 为什么需要 Normalization？

    深度网络训练中的两个核心问题：

    • Internal Covariate Shift：每层输入的分布随训练变化，迫使后续层不断适应
    • 梯度消失/爆炸：深层网络中梯度在反向传播时容易衰减或放大

    2. Batch Normalization

    BatchNorm 在 batch 维度上归一化：

    [ \hat{x}_i = \frac{x_i - \mu_B}{\sqrt{\sigma_B^2 + \epsilon}}, \quad y_i = \gamma \hat{x}_i + \beta ]

    2.1 BatchNorm 的问题

    • 依赖 batch size：小 batch 下统计量不稳定
    • 训练/推理不一致：推理时使用移动平均，行为有差异
    • 序列建模不友好：RNN/Transformer 中每个时间步需要独立统计量

    3. Layer Normalization

    LayerNorm 在特征维度上归一化，对每个样本独立计算：

    [ \mu = \frac{1}{d}\sum_{i=1}^d x_i, \quad \sigma^2 = \frac{1}{d}\sum_{i=1}^d (x_i - \mu)^2 ]

    [ \hat{x}_i = \frac{x_i - \mu}{\sqrt{\sigma^2 + \epsilon}}, \quad y_i = \gamma_i \hat{x}_i + \beta_i ]

    LayerNorm 不依赖 batch size，训练和推理行为一致，适合序列建模。

    4. RMSNorm

    RMSNorm 是 LayerNorm 的简化版，去掉均值中心化：

    [ \text{RMS}(x) = \sqrt{\frac{1}{d}\sum_{i=1}^d x_i^2}, \quad \hat{x}_i = \frac{x_i}{\text{RMS}(x) + \epsilon} \cdot \gamma_i ]

    4.1 为什么去掉均值也行？

    • 均值中心化对性能影响很小
    • RMS 统计量已经足够稳定分布
    • 减少计算量约 10-15%

    5. Pre-Norm vs Post-Norm

    [ \text{Post-Norm: } \quad x_{l+1} = \text{LN}(x_l + F(x_l)) ] [ \text{Pre-Norm: } \quad x_{l+1} = x_l + F(\text{LN}(x_l)) ]

    5.1 梯度角度的分析

    Post-Norm 中，梯度经过 LN 才能到残差路径。Pre-Norm 中，梯度直接传播，减少梯度消失，不需要 learning rate warmup，更深网络更稳定。

    6. DeepNorm

    微软提出用于训练极深 Transformer（1000+ 层）：

    [ x_{l+1} = x_l + \alpha \cdot F(\text{LN}(x_l)) ]

    其中 $\alpha$ 是一个小于 1 的常数，核心思想是在初始化时约束残差分支的方差。
