    在线GPU练习平台汇总

    在线 GPU 练习平台汇总：免费白嫖 + 低价租赁

    更新时间：2026-06-30 状态：工具收集笔记，按”免费 → 低价租赁”分层 用途：配合 AI-Infra 七阶段学习，随时上手实操 GPU 环境，不用自己买显卡 关联：本目录 阶段一 到 阶段七 笔记

    一句话结论

    学习阶段优先用免费平台（Google Colab / Kaggle）白嫖 GPU 跑小模型实验；到了阶段三/五/六需要大显存跑 14B 模型时，花几块钱在 AutoDL 按小时租 A10/A100 最划算。 国内用户首选 Kaggle（免翻墙）+ AutoDL（支付宝直付）；海外用户首选 Google Colab。

    一、免费平台（白嫖算力）

    1.1 Google Colab（海外首选，但需翻墙）

    网址：https://colab.research.google.com/

    维度免费版Pro 版（$9.99/月）
    GPUT4 15GB（随机分配）T4/V100/A100 可选
    时长每次最多 12 小时，断开后回收24 小时
    内存~12GB~32GB
    磁盘~100GB 临时~200GB
    网络需翻墙同左

    特点：

    • Google 出品，Jupyter Notebook 在线版，开箱即用
    • 自带 PyTorch / TensorFlow / CUDA 环境，不用自己装
    • 可直接挂载 Google Drive 存数据
    • 最大痛点：国内需翻墙，连接不稳定

    适合：学习 PyTorch 基础、跑小模型训练、阶段三验证 CUDA 环境

    使用技巧：

    # 查看分配的 GPU
!nvidia-smi

# 安装额外依赖
!pip install vllm

# 挂载 Google Drive
from google.colab import drive
drive.mount('/content/drive')

    1.2 Kaggle Notebooks（国内可直连，推荐）

    网址：https://www.kaggle.com/code

    维度免费版
    GPUT4 x2（双卡，30GB 显存）
    TPUv3-8（可选）
    时长每周 30 小时 GPU，每次最多 12 小时
    内存16GB
    磁盘20GB 临时 + 100GB 数据集存储
    网络国内可直连，无需翻墙

    特点：

    • Kaggle 出品（Google 旗下），Jupyter Notebook 环境
    • 国内直连，不用翻墙，这是相对 Colab 的最大优势
    • 双 T4 GPU，显存 30GB，能跑 7B 模型
    • 自带常用数据集，一键加载
    • 注册只需邮箱，不需要海外手机号

    适合：国内用户首选免费平台，阶段三/五跑 7B 模型实验

    使用技巧：

    # 开启 GPU：Settings → Accelerator → GPU T4 x2
!nvidia-smi

# 查看 GPU 数量
import torch
print(torch.cuda.device_count())  # 2

    1.3 国内免费平台（备选）

    平台网址特点
    阿里天池实验室https://tianchi.aliyun.com/阿里云提供，稳定性好，GPU 资源有限
    百度 AI Studiohttps://aistudio.baidu.com/百度飞桨生态，免费 V100 100 小时/周
    腾讯 Cloud Studiohttps://cloudstudio.net/腾讯云，CPU 为主，GPU 少
    MindSpore 华为云https://www.mindspore.cn/华为昇思生态，免费 Ascend 算力

    国内免费平台 GPU 资源通常较少，适合轻量实验。大模型训练建议用下面的低价租赁平台。

    二、低价租赁平台（按小时计费，几块钱起步）

    2.1 AutoDL（国内首选，强烈推荐）

    网址：https://www.autodl.com/

    为什么是首选：

    • 中国大陆最大 GPU 算力租赁平台
    • 支付宝/微信直接付款，无需海外卡
    • 镜像市场有大量预装环境（PyTorch、vLLM、Stable Diffusion 等），开箱即用
    • 秒级计费，用完关机不计费

    价格参考（2026 年，可能有波动）：

    GPU 型号显存价格适合
    RTX 309024GB~1.2 元/小时阶段三/五，跑 7B-14B 模型
    RTX 409024GB~2.0 元/小时阶段五，推理+训练
    A100 40G40GB~3.5 元/小时阶段五/六，14B 量化模型
    A100 80G80GB~6.0 元/小时70B 模型
    多卡 A10080GB×N按卡数倍增多机多卡训练

    使用流程：

    # 1. 注册充值（支付宝）
# 2. 选择显卡 → 选择镜像（如 PyTorch 2.1 + CUDA 12.1）
# 3. 自动创建实例，获得 SSH 地址和密码
# 4. SSH 连接
ssh -p 12345 root@connect.xxx.autodl.com
# 5. 或用 JupyterLab（网页直接写代码）
# 6. 用完记得「关机」，否则持续计费！

    省钱技巧：

    • 关机不计费：用完点”关机”，只收几毛钱/小时的磁盘费
    • 镜像市场：选预装好 vLLM/PyTorch 的镜像，省安装时间
    • 数据盘持久化：关机后数据不丢，下次开机接着用
    • 无卡模式：只想写代码不跑 GPU，开无卡模式（0.1 元/小时）

    适合：阶段三到阶段六所有实操任务，跑 14B 模型性价比最高

    三、海外云平台（按需使用）

    3.1 Lightning AI Studio（原 PyTorch Lightning）

    网址：https://lightning.ai/studio

    维度免费版Pro 版
    GPU每月 4 小时 T4按需购买
    CPU每月 22 小时-
    存储10GB 持久更大
    特点PyTorch 官方生态，和 Lightning 框架深度集成 

    3.2 AWS / GCP / Azure（企业级，贵）

    平台GPU 实例价格特点
    AWSP4d（8×A100 40G）~$32/小时企业级，贵
    GCPA2（A100 40G/80G）~$3.6/小时灵活配置
    AzureNC 系列类似 AWS企业合规

    不推荐个人学习用，价格太高。企业项目可申请云厂商额度。

    3.3 Vast.ai / RunPod（社区算力市场）

    平台特点价格
    Vast.ai个人出租闲置 GPU，最便宜~$0.2-0.5/小时（RTX 3090）
    RunPod社区+专业混合~$0.4-0.8/小时

    价格低但稳定性不如专业平台，适合跑不怕中断的任务。

    四、平台选型决策树

    你的需求是什么？
│
├─ 只是想学 PyTorch 基础，跑小模型
│   ├─ 能翻墙 → Google Colab（免费 T4）
│   └─ 不能翻墙 → Kaggle（免费双 T4，国内直连）
│
├─ 要跑 7B 模型（需 16GB+ 显存）
│   ├─ 免费 → Kaggle（双 T4 30GB）
│   └─ 付费 → AutoDL RTX 3090（24GB，1.2 元/小时）
│
├─ 要跑 14B 量化模型（需 10GB+ 显存）
│   └─ AutoDL RTX 3090/4090（24GB，1.2-2 元/小时）
│
├─ 要跑 14B 原始模型 / 70B 量化（需 28GB+ 显存）
│   └─ AutoDL A100 40G/80G（3.5-6 元/小时）
│
├─ 多机多卡训练（阶段六项目）
│   ├─ AutoDL 多卡 A100
│   └─ 学校/公司集群
│
└─ 长期训练（几天不断）
    └─ AutoDL 包周/包月套餐（比按小时更省）

    六项目二（3 节点）3 台 AutoDL + K8s~100 元

    总预算：完整 7 阶段实操约 200-400 元（不含 K8s 集群月费）。如果只用免费平台，阶段一/二/三/五部分能覆盖，但 14B 模型部署基本要付费。

    六、省钱核心原则

    1. 用完立即关机：AutoDL 关机不计费，只收磁盘费（几毛/小时）
    2. 先写代码后开机：用无卡模式或本地写好脚本，再开 GPU 跑
    3. 善用镜像市场：选预装环境，省安装时间（时间=钱）
    4. 数据持久化：数据存数据盘，关机不丢，避免重复上传
    5. 包周/包月更省：长期训练买套餐，比按小时便宜 30-50%
    6. 免费额度先用完：Kaggle 每周 30 小时 GPU 先用，不够再付费

    七、注册与上手顺序建议

    1. 注册 Kaggle（免费，邮箱即可）        → 阶段一/二/三 基础实验
2. 注册 AutoDL（免费，支付宝充值 10 元）→ 阶段三/五/六 大模型实操
3. 注册 Google Colab（可选，需 Google 账号）→ 备用免费资源
4. 买阿里云 2 核 4G（阶段一 Linux 练习）  → 50 元/月

    八、踩坑记录

    待实际使用后补充。预留常见坑方向：

    • 坑1（预期）：Colab 免费版 12 小时强制断开 → 重要训练做 checkpoint
    • 坑2（预期）：Kaggle 数据上传慢 → 用 Kaggle 自带数据集
    • 坑3（预期）：AutoDL 忘记关机 → 设闹钟提醒，或用自动关机脚本
    • 坑4（预期）：镜像环境不匹配 → 选镜像时确认 CUDA/PyTorch 版本
    • 坑5（预期）：Vast.ai 中途被回收 → 只跑可重试的任务
