    算法刷题在线练习平台

    算法刷题在线练习平台

    更新时间：2026-06-30 状态：工具收集笔记 用途：面试算法题刷题平台汇总，配合字节/大厂面试准备 关联：面试经历/agent开发/2026-06-15-字节.md（字节面试算法题）

    一句话结论

    大厂面试算法首选 LeetCode 力扣（题库最全+大厂原题多）+ 牛客网（剑指 Offer+公司真题卷）。基础薄弱想系统学用 AcWing，想冲竞赛用 Codeforces。刷题顺序：先 LeetCode Hot 100 → 剑指 Offer → 公司真题卷。

    一、平台对比总览

    平台网址定位难度中文推荐度
    LeetCode 力扣leetcode.cn大厂面试首选中-难✅⭐⭐⭐⭐⭐
    牛客网nowcoder.com国内面试向中✅⭐⭐⭐⭐⭐
    AcWingacwing.com算法系统学习中-难✅⭐⭐⭐⭐

    CodeWarscodewars.com趣味入门易-中❌⭐⭐

    二、LeetCode 力扣（首选，必刷）

    网址：https://leetcode.cn/

    2.1 为什么是首选

    • 题库最全：3000+ 题，覆盖所有常见算法题型
    • 大厂原题多：字节/腾讯/阿里面试算法题基本都能找到
    • 题解社区活跃：每题都有中英文题解，含多种解法
    • 企业题库：有按公司整理的高频题（LeetCode Premium 功能，但社区有免费整理版）

    2.2 推荐刷题顺序

    顺序题单题数目标
    1LeetCode Hot 100100 题高频必刷，覆盖核心题型
    2面试经典 150 题150 题官方精选，Hot 100 的补充
    3剑指 Offer75 题经典面试题（牛客上也有）
    4字节跳动专题~200 题投字节必刷
    5每日一题持续保持手感

    2.3 高频题型分布（Hot 100 统计）

    题型占比代表题
    数组/哈希表25%两数之和、子数组和为 K
    双指针/滑动窗口15%三数之和、最长无重复子串
    链表15%反转链表、合并有序链表
    二叉树/DFS20%LCA、层序遍历
    动态规划15%最长递增子序列、编辑距离
    其他（栈/堆/图）10%有效括号、拓扑排序

    2.4 使用技巧

    • 先按题型刷，不要随机刷（同题型集中突破效率高）
    • 先简单后中等，Hard 视目标公司决定是否刷
    • 每题先想 15 分钟，想不出看题解，理解后自己重写一遍
    • 用 Python 刷（AI 方向面试通常用 Python，代码短效率高）

    2.5 LeetCode 旗下赛道板块 + LeetXXX 衍生平台

    LeetCode 主站内还有多个专项赛道（非独立域名，共享主站账号）：

    赛道定位
    LeetDatabaseSQL 专项
    LeetShellShell 脚本
    LeetConcurrency多线程并发编程
    LeetPandasPandas 数据处理
    LeetDesign系统/OOD 设计

    此外还有一批独立第三方 LeetXXX 刷题站（LeetGPU、LeetCUDA、LeetSQL 等），尤其 LeetGPU/LeetCUDA 是 AI Infra 推理优化方向的核心刷题平台。

    📌 详细整理见本目录 LeetXXX系列刷题平台整理.md

    三、牛客网（国内面试必备）

    网址：https://www.nowcoder.com/

    3.1 核心优势

    • 剑指 Offer 原题：75 道经典面试题，国内面试覆盖率极高
    • 公司真题卷：字节/腾讯/阿里/美团等历年笔试真题
    • 笔试模拟：限时模拟真实笔试环境
    • 讨论区：面试面经分享

    3.2 推荐题单

    题单说明
    剑指 Offer75 题，必刷，面试原题率极高
    字节跳动 2024 校招字节笔试真题
    腾讯 2024 校招腾讯笔试真题
    算法笔试全真模拟限时模拟，练手感

    3.3 配合 LeetCode 使用

    • LeetCode 刷题型基础 → 牛客刷公司真题卷
    • 牛客的剑指 Offer 和 LeetCode 的剑指 Offer 题目相同，选一个平台刷即可

    四、AcWing（系统学算法）

    网址：https://www.acwing.com/

    4.1 适合谁

    • 算法基础薄弱，想系统学习
    • 目标是蓝桥杯/ACM 竞赛
    • 喜欢有课程引导的学习方式

    4.2 核心内容

    课程内容价格
    算法基础课排序、二分、DP、图论等基础付费（有免费试听）
    算法提高课进阶算法付费
    蓝桥杯辅导课竞赛专项付费

    4.3 题库特点

    • 题目按算法分类（不像 LeetCode 按难度）
    • 每题有视频讲解
    • 适合”学一个算法刷一类题”

    五、其他平台简评

    5.1 Codeforces（国际竞赛）

    网址：https://codeforces.com/

    • 国际顶级算法竞赛平台
    • 题目难度高，适合 ACM 选手
    • 有定期比赛（Round），可提升排名
    • 不适合面试准备，除非目标是对冲基金/量化

    5.2 HackerRank（外企面试）

    网址：https://www.hackerrank.com/

    • 国际化平台，外企（Google/Meta/Amazon）面试常用
    • 有企业认证考试
    • 题目偏工程化（不只是纯算法）
    • 投外企必刷，投国内大厂用 LeetCode 即可

    5.3 CodeWars（趣味入门）

    网址：https://www.codewars.com/

    • 游戏化设计，解题升级
    • 题目小巧，适合碎片时间
    • 多语言支持（Python/Go/JS 等）
    • 适合零基础入门，进阶后转 LeetCode

    六、刷题策略建议

    6.1 按目标公司

    目标平台题单题数
    字节跳动LeetCode字节专题 + Hot 100~300
    腾讯/阿里LeetCode + 牛客Hot 100 + 公司真题~200
    外企LeetCode + HackerRankHot 100 + HackerRank 认证~200
    中小公司LeetCodeHot 100100

    6.2 按时间规划

    时间目标每日题量
    3 个月大厂面试3-5 题/天
    1 个月冲刺面试5-8 题/天（重质量）
    1 周临时突击只刷 Hot 100 高频题

    6.3 按语言选择

    语言适合场景刷题体验
    PythonAI 方向面试（推荐）代码短，写得快
    Java后端面试代码长但规范
    Go云原生面试简洁，适合并发题
